# wechat-layout-publisher v1.3.0

这一版来自一次完整的公众号真实制作验收，重点解决开局模式遗漏、排版异常留白、配图位置不合理、长截图疲劳和视觉重复等实战问题。

## 主要更新

- 开局固定提供杂乱资料、初稿文案、发布定稿三种内容处理模式，以及可复制版和直接草稿箱两种交付方式。
- 初稿文案拥有独立的诊断、修订、定稿和修改说明流程。
- 新增统一排版检查器，自动拦截可见 Markdown 分隔符、异常留白和首节配图过晚。
- 图片、SVG、长截图、表格、引用、大卡片、矩阵和步骤块统一计入强视觉密度。
- 自动拦截连续重视觉、连续长截图，以及数据卡片后紧接同义图表、步骤卡后紧接同义流程图等重复表达。
- 图片计划新增首节视觉锚点、语义签名、真实尺寸和截图裁切策略。
- 移动端视觉 QA 增加整页截图重复平铺检测，并同时绑定文章和图片计划。
- 微信托管图回拉失败时，安全复用经过哈希验证的本地母版，不放宽 SSRF 防护。
- 回归测试增至 42 项，并加入真实 Ollama 排版问题夹具。

## 下载

- `wechat-layout-publisher.zip`：始终保持相同名称，供最新版直链使用。
- `wechat-layout-publisher-v1.3.0.zip`：固定版本归档。
- 对应 `.sha256` 文件用于校验下载完整性。

纯净包不包含 `node_modules`、公众号凭据、测试过程截图或文章生成物。解压后请在 `wechat-layout-publisher/scripts` 中运行 `npm ci --omit=dev`。
