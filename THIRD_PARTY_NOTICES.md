# Third-Party Review

The design research for this Skill included [`op7418/guizang-social-card-skill`](https://github.com/op7418/guizang-social-card-skill), which is distributed under AGPL-3.0. This repository is also distributed under AGPL-3.0. It does not bundle source files, templates, or assets from that repository; its design guidance informed parts of the visual-planning approach.

The current implementation, templates, and documentation were written for this project. A release audit found no identical nontrivial lines between the distributable package and the referenced Skill.

An earlier local `wechat-publisher` Skill was also studied for workflow concepts such as draft-box publishing. No files from that reference package are distributed here.
