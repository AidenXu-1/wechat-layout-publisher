# wechat-layout-publisher v1.2.0

这一版把公众号文章的双模式交付、正文保护和首图视觉要求收紧为可验证的正式闸门。

## 主要更新

- 稳定保留正文与重写正文两种工作模式。
- 可复制版本保持安全默认，只有显式授权才创建公众号草稿。
- 最终文章、图片计划与移动端视觉 QA 证据必须相互对应。
- 首图必须是模型生成的 `900 × 383` 成品头图，文章 H1 需要自然融入画面。
- 删除自动 SVG 标题覆盖降级路径，阻止纯图片或粗糙贴字首图通过检查。
- 回归测试增至 32 项。

## 下载

- `wechat-layout-publisher.zip`：始终保持相同名称，供最新版直链使用。
- `wechat-layout-publisher-v1.2.0.zip`：固定版本归档。
- 对应 `.sha256` 文件用于校验下载完整性。

纯净包不包含 `node_modules`、凭据和生成文件。解压后请在 `wechat-layout-publisher/scripts` 中运行 `npm ci --omit=dev`。
